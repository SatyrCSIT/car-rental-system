# 🚗 Car Rental System 🎉

**Car Rental System** โปรเจกต์ระบบเช่ายานพาหนะออนไลน์ที่พัฒนาขึ้นในปี 2566 เพื่อให้บริการเช่ายานพาหนะสำหรับผู้ใช้ทั่วไป (ลูกค้า) ระบบนี้พัฒนาด้วย PHP, MySQL และใช้ Bootstrap สำหรับส่วนติดต่อผู้ใช้ (UI) เพื่อให้ใช้งานง่ายและ responsive รองรับทั้งการสมัครสมาชิก การจัดการยานพาหนะ การเช่ารถ และการส่ง feedback จากลูกค้า

---

## ✨ Features

### User
- ✅ สมัครสมาชิกและล็อกอิน
- ✅ ส่ง feedback หลังใช้บริการ
- ✅ ดูข้อมูลยานพาหนะและเช่ารถ

### Admin
- ✅ ดูและจัดการ feedback จากลูกค้า (รวมถึงการลบ feedback)
- ✅ จัดการข้อมูลยานพาหนะ (เพิ่ม/แก้ไข/ลบ)
- ✅ จัดการสถานะยานพาหนะ (รถว่างให้เช่า/รถไม่ว่างให้เช่า)

---

## Technologies 💻

- **Backend**: PHP 8.2.12
- **Database**: MySQL/MariaDB 10.4.32
- **Frontend**: Bootstrap 5.3.0, jQuery 3.6.0
- **Server**: Apache (ผ่าน XAMPP)
- **SQL Dump**: `project_carrent.sql`
